Virtours

VIRTUAL REALITY

Virtours is a VR trip planner which allows users to choose where they want to travel, suggests attractions, helps plan each trip in detail and allows users to book hotels.

Introduction

There are lot of trip planning methods and options online to help people plan their trips, But with the invasion of latest technologies such as VR/AR have lot of potential in revolutionizing the ways these trips are planned and experienced.

User Interviews

I conducted user interviews to learn more about people’s trip planning process and which tools they normally used. I enquired about how people choose their travel destinations, the biggest obstacles when planning trips, and the biggest concerns when traveling. Here are some of the key take-aways from the results:

Trip Planning Process

While conducting the the interviews, I observed people’s planning process and classified their steps into separate stages.

Final Design

Landing page

• Select places from a list and select the time for the trip.

• Explore the place in the next step.

Destination page

• There are few, but critical pieces of information, such as the destination name, live weather, a few lines about the destination and top sites to see.

Site details page

• See the location data with 360º panorama photos.

• Virtually travel to the specific location and have a rough overview of the site.

Select must see places

• Users can pin various locations to spend time, which are automatically added to the trip itinerary.  

• Users have access to different pins from various other users, which can be explored virtually before actually traveling.

Pick hotels

• Users have a map view of available hotels.

• There is an option to view the hotel in VR, which will give users an added layer of understanding of the hotel’s information.

• Users can directly scroll other hotels and see the hotel’s description from a card-list on the right side.
